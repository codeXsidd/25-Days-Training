# 🗓️ Day 10: 31st March

## 🎯 Focus: Object-Oriented Programming (Inheritance)

Today's session was dedicated to exploring and implementing various types of inheritance in Java, culminating in a real-world application of these concepts in a project.

---

## 🏗️ Core OOP Concepts Explored

### 1. Single Inheritance
- **Concept:** One subclass inherits from one superclass.
- **File:** `Single_Inheritance.java`

### 2. Hierarchical Inheritance
- **Concept:** Multiple subclasses inheriting from a single superclass.
- **File:** `Hierarchical_Inheritance.java`

### 3. Multi-Level Inheritance
- **Concept:** A subclass inherits from a superclass that is also a subclass itself.
- **File:** `MultiLevel_Inheritance.java`

---

## 🛠️ Project Implementation

### 4. Ride-Booking App
- **Description:** A simple simulation of a ride-booking marketplace using inheritance to model different types of vehicles and pricing strategies.
- **Conceptual Application:** Demonstration of polymorphism and shared superclass behavior.
- **File:** `Ride_Booking_App.java`
