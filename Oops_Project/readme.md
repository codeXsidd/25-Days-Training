# 🚀 Final Project: Mini Social Media App

## 🎯 Focus: Consolidation of CORE Java & OOP Principles

This project serves as the culmination of the 23-day training, leveraging foundational concepts in Java and advanced Object-Oriented Design to build a functional, modular application.

---

## 🏗️ Project Overview

The **Mini Social Media App** is a simulated platform where users can interact through posts, comments, and relationships. It is designed to demonstrate key Java functionalities in a real-world scenario.

---

## ✨ Features Implemented

*   **User Management:** Create, update, and manage unique user profiles.
*   **Content Interaction:** Post creation, commenting, and nested interactions.
*   **Relationship Mapping:** Modeling follower/friendship networks using Java collections.
*   **Modular Architecture:** Separation based on logical components.

---

## 🛠️ Java Concepts Applied

| Concept | Implementation Detail |
| :--- | :--- |
| **Encapsulation** | Private fields with public getters/setters for data integrity. |
| **Inheritance** | Shared behavior between different types of posts/notifications. |
| **Abstraction** | Interfaces used to define generic interaction behaviors. |
| **Polymorphism** | Dynamic handling of various post types during feed rendering. |
| **Collections** | Efficient management of users and content using `ArrayList`, `HashMap`, etc. |

---

## 🚀 How to Run

Compile and execute the main Java class:

```bash
javac Mini_Social_Media_App.java
java Mini_Social_Media_App
```
