# 🗓️ Day 11: 1st April

## 🎯 Focus: Polymorphism & Sub-array Optimization

Training today shifted to understanding Polymorphism, a key pillar of Object-Oriented Programming, alongside a complex array problem.

---

## 🧠 Algorithmic Problem

### 1. Longest Mountain in Sub-array
- **Description:** Locate the longest continuous sub-array that satisfies the "mountain" property (increases then decreases).
- **Complexity:** O(N) linear scan approach.
- **File:** `Longest_Mountain_SubArray.java`

---

## 🏗️ Core OOP Concepts Explored

### 2. Method Overloading (Compile-time Polymorphism)
- **Concept:** Multiple methods with the same name but different signatures within the same class.
- **File:** `Polymorphism(Method_overloading).java`

### 3. Method Overriding (Run-time Polymorphism)
- **Concept:** Re-defining a method in a subclass that was already defined in the parent class.
- **File:** `Polymorphism(Method_overriding).java`
