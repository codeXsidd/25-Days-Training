# 🗓️ Day 16: 9th April

## 🎯 Focus: Java Collections (Maps & Sets) & Frequency Sorting

Today's session was dedicated to exploring the Java Collections Framework in detail, specifically focusing on various implementations of `Set` and `Map`, and solving string frequency challenges.

---

## 🏗️ Core Concepts Explored

### 1. The Map Interface
- **Implementations:** `HashMap` (Unordered), `LinkedHashMap` (Insertion Order), and `TreeMap` (Sorted Order).
- **Key Discovery:** Understanding when to use which implementation based on performance and ordering requirements.

### 2. The Set Interface
- **Implementations:** `HashSet` (Unique & Unordered), `LinkedHashSet` (Unique & Ordered), and `TreeSet` (Unique & Sorted).

### 3. Frequency Sorting
- **Logic:** Building frequency maps for characters and sorting the map/string based on those frequencies.
